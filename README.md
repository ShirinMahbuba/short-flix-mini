# Short-Flix Mini - Netflix Style Short Video Platform

A fully functional mini short-video platform built with Next.js 14 (App Router), TypeScript and Tailwind CSS.

### Features
- GET /api/shorts endpoint with 8 hardcoded short videos
- Responsive Netflix-style video grid
- Search by title
- Like/favorite button (saved in localStorage)
- Click to open full-screen modal video player with controls
- All videos play perfectly (using reliable public test video sources)

### Tech Stack
- Next.js 14 + TypeScript
- Tailwind CSS
- Vercel (frontend + API in single deployment)
